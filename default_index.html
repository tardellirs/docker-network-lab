Your personal server is up and running!

This is the default page of your Nginx web server. You can edit it to build your own website.

How to edit this page

In your terminal, run:

nano /var/www/html/index.html

Or if you prefer Vim:

vim /var/www/html/index.html

Quick tips

Test connectivity: ping student02

View your network config: ip addr show

Capture packets: tcpdump -i eth0

Scan the network: nmap 172.20.0.0/24
